Understanding Color Models

🖼️

How the Image Picker Works

The image is drawn on an HTML Canvas. When you hover or click, the tool reads the exact RGBA values of that pixel using getImageData(). No data is sent to any server — everything runs locally in your browser.

🔢

What is a HEX Code?

A HEX code is a six-character hexadecimal number for RGB color, prefixed with #. Each pair encodes Red, Green or Blue from 00 (0) to FF (255). Example: #FF5733 is a warm orange-red.

⚖️

RGB vs HSL

RGB controls light channels directly. HSL uses Hue (0–360°), Saturation and Lightness — easier for adjustments. To lighten a color by 10%, just increase the L value by 10.

🏭

RAL Colors

RAL is a European standard for paints and coatings. RAL Classic has 213 codes. RAL Design System+ extends this with 1,625+ colors organized by hue and lightness — ideal for architecture and interior design.

🎨

Pantone & NCS

Pantone is the global print and brand identity standard. NCS is based on human color perception, widely used in Scandinavian design. Both use HEX approximations for screen display.

Frequently Asked Questions

How do I pick a color from an image?
Click "Upload Image" or drag a photo onto the upload area. Once loaded, hover over any part of the image to preview its color. Click to lock in that color and see the full analysis.
Does my image get uploaded to a server?
No. The image is processed entirely in your browser using the HTML Canvas API. No image data is sent to any server. Your photos remain private.
What image formats are supported?
Any format your browser supports — JPG, PNG, WebP, GIF, BMP, SVG. There is no size limit, though very large images may take a moment to load.
What color standards are supported?
Five standards: CSS Named Colors (W3C), RAL Classic (industrial), RAL Design System+ (architectural), Pantone Solid Coated (brand/print), and NCS Natural Color System. Select with the radio buttons above the color result.
Can I pick multiple colors from the same image?
Yes — simply click different pixels in the image. Each click updates the color result. Use the HEX chip to copy each color before clicking the next one.
Can I also pick a color from my screen?
Yes — scroll down to the EyeDropper button, or use the Pick from Screen tool. This requires Chrome or Edge browser.
