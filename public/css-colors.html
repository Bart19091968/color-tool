About CSS Named Colors

CSS named colors are a set of 148 color keywords defined by the W3C in the CSS Color specification. You can use them directly in HTML and CSS:

color: dodgerblue; or background-color: tomato;

The list has evolved over time — CSS1 included 16 colors, CSS3 extended this to 147, and CSS4 added rebeccapurple (named after Rebecca Meyer). All modern browsers support the full list.

CSS color names are case-insensitive. They cover a wide range from aliceblue (#F0F8FF) to yellowgreen (#9ACD32).
