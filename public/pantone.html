About Pantone Colors

Pantone is the world's leading color standard system, used by designers, brands and manufacturers globally for print, fashion, product design and digital work. Each Pantone color has a unique code (e.g. Pantone 186 C) and a standardized appearance.

This database includes 338 Pantone Solid Coated colors with their HEX screen approximations. Physical Pantone colors are defined for ink on coated paper — HEX values are widely-used approximations for screen display.

Frequently Asked Questions
What is a Pantone color code?
A Pantone code is a unique identifier in the Pantone Matching System (PMS) — for example, Pantone 186 C. The letter suffix indicates the paper type: C = coated, U = uncoated, M = matte. The code ensures consistent color reproduction across different printers and manufacturers worldwide.
Are Pantone HEX values exact?
No. Pantone colors are defined for physical pigments on paper. HEX values are widely-published screen approximations. The actual appearance depends on your monitor calibration. For printing, always use the official Pantone Matching System.
What Pantone color is red #C8102E?
Pantone 186 C is the closest match for #C8102E — a vivid red used in many national flags and brand identities. You can verify this by entering the HEX in Coloralyze with the Pantone standard selected.
How do I find the Pantone color of a logo?
Upload a screenshot or photo of the logo on the Pick from Image page, select the Pantone standard, and click the logo color. Coloralyze returns the closest Pantone name and code. For brand-critical work, always verify with a physical Pantone fan deck.

Yes. Copy the HEX code from Coloralyze, open Canva, select any element, click the color swatch, and paste the HEX in the input field. Canva Pro users can also save brand colors directly. Try Canva free →
