How to Use HEX Color Codes in Canva — Step by Step

Published 2025-02-01 · Coloralyze

Canva is used by over 170 million people worldwide, and one of its most powerful — yet underused — features is the ability to enter exact HEX color codes. Here is everything you need to know.

Step 1: Find your HEX color code

Before opening Canva, you need the HEX code of the color you want to use. If you do not have it:

  • From your screen: Use Coloralyze Pick from Screen — click anywhere on your screen to capture any color
  • From a photo: Use Coloralyze Pick from Image — upload a photo and click any pixel
  • From a brand guide: Most brand guides list HEX codes directly

A HEX code looks like this: #7950F2 (a vivid purple). The # sign is required in most applications.

Step 2: Copy the HEX code

In Coloralyze, click the HEX chip below the color swatch. It turns green and shows "✓ Copied!" — the code is now in your clipboard.

Step 3: Open the color picker in Canva

In your Canva design:

  • Select any element (text, shape, background, etc.)
  • Click the color swatch that appears in the toolbar at the top
  • The Canva color picker opens

Step 4: Enter the HEX code

  • At the bottom of the color picker, you will see a text field with a HEX code (default is the current color)
  • Click the field, select all the text, and paste your copied HEX code
  • Press Enter or click away — the color updates instantly

Step 5: Save as a brand color (optional)

If you use this color regularly, save it to your Canva palette:

  • In the color picker, click Add to Brand Kit (Canva Pro) or the + button next to your document colors
  • Your color appears in the palette and is reusable across all your designs

Bonus: Use color names in Canva

Once you know the Pantone, RAL or NCS name of your color from Coloralyze, you can reference it in your brand documents. Include both the HEX code (for digital use in Canva) and the Pantone code (for print) in your brand guidelines.
