How to Use HEX Color Codes in Adobe Express — Step by Step

Published 2025-04-19 · Coloralyze

Adobe Express is used by millions of designers, marketers and creators worldwide, and one of its most powerful — yet underused — features is the ability to enter exact HEX color codes. Here is everything you need to know.

Step 1: Find your HEX color code

Before opening Adobe Express, you need the HEX code of the color you want to use. If you do not have it:

  • From your screen: Use Coloralyze Pick from Screen — click anywhere on your screen to capture any color instantly
  • From a photo: Use Coloralyze Pick from Image — upload a photo and click any pixel to extract its exact color
  • From a brand guide: Most brand guides list HEX codes directly alongside Pantone and CMYK values

A HEX code looks like this: #FA2719 (Adobe red). The # sign is required in most applications.

Step 2: Copy the HEX code

In Coloralyze, click the HEX chip below the color swatch. It turns green and shows "✓ Copied!" — the code is now in your clipboard, ready to paste.

Step 3: Open the color picker in Adobe Express

In your Adobe Express design:

  • Select any element — text, shape, background, or image overlay
  • Click the color swatch that appears in the toolbar on the left or top
  • The Adobe Express color picker panel opens

Step 4: Enter the HEX code

  • In the color picker, locate the HEX input field — it shows the current color's code (e.g. #000000)
  • Click the field, select all existing text, and paste your copied HEX code
  • Press Enter — the color updates instantly across your design element
  • Tip: you can also type the 6-digit code without the # sign and Adobe Express will add it automatically

Step 5: Save as a brand color

Adobe Express makes it easy to save colors for reuse across all your projects:

  • In the color picker, click the + button next to "Your colors" to save the color to your personal palette
  • Adobe Express Premium users can save colors directly to a Brand Kit — making the exact HEX available across all team members and templates
  • Saved colors appear at the top of the color picker for one-click access in every future design

Step 6: Apply to multiple elements at once

Adobe Express has a powerful theme feature that lets you swap all colors in a design simultaneously. Once you have your brand HEX saved, you can:

  • Click Themes in the left panel
  • Create a custom theme using your saved HEX colors
  • Apply it across your entire design in one click — perfect for maintaining brand consistency

Bonus: Use color names from Coloralyze in your brand guide

Once Coloralyze identifies your color across Pantone, RAL, NCS and CSS standards, you have everything you need for a professional brand guide:

  • Use the HEX code for all digital work in Adobe Express
  • Use the Pantone code for print materials, packaging and signage
  • Use the RAL code for physical products, coatings and interior design

This gives your brand a single source of truth across every medium.
