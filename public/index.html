Understanding Color Models

🔢

What is a HEX Code?

A HEX code is a six-character hexadecimal number for RGB color, prefixed with #. Each two-character pair encodes Red, Green or Blue from 00 (0) to FF (255). Example: #FF5733 is a warm orange-red.

⚖️

RGB vs HSL

RGB controls light channels directly. HSL uses Hue (0–360°), Saturation (0–100%) and Lightness (0–100%) — much easier for programmatic adjustments like making a color 20% lighter.

RAL Colors

RAL is a European standard for paints and coatings. RAL Classic has 213 codes (e.g. RAL 3000 — Flame Red). RAL Design System+ extends this to 1,625+ colors by hue and lightness.

Pantone & NCS

Pantone is the global print & brand identity standard. NCS (Natural Color System) is based on human color perception. Both use HEX approximations for screen display.

🌈

Complementary Colors

A complementary color is directly opposite on the color wheel. Triadic colors are 120° and 240° apart on the HSL wheel — both form the basis of professional color systems.

Frequently Asked Questions

How does the EyeDropper work?
Click "Pick from Screen". Your cursor becomes a crosshair. Click any pixel on your screen — including outside the browser window. The tool captures the exact color at that point. Requires Chrome 95+ or Edge 95+.
What is a HEX color code?
A HEX code is a six-digit hexadecimal number for an RGB color, starting with #. Each pair represents Red, Green and Blue (00–FF). For example, #FF5733 is a warm orange-red.
What color standards are supported?
Five standards: CSS Named Colors (W3C), RAL Classic (industrial), RAL Design System+ (architectural), Pantone Solid Coated (brand/print), and NCS Natural Color System. Select with the radio buttons above the color tool.
What is the difference between RGB and HSL?
RGB mixes red, green and blue channels. HSL describes Hue (0–360°), Saturation and Lightness. HSL is easier for design work — to make a color lighter, simply increase the L value.
How do I use my color in Canva?
Click the HEX chip to copy your code. In Canva, select any element, click its color swatch in the toolbar, and paste your HEX in the color input field. Canva is free at canva.com.
Can I pick a color from an image instead?
Yes — use the Pick from Image tool. Upload any photo and click any pixel to extract its exact color.
